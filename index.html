Project Example #1

Evidence-based practice is a collaboration between the Instructional Design Team and the Health Science Librarians.

Why

I included this tutorial because it is build without using a rapid e-learning tool - using reveal.js and Qualtrics instead. It is responsive and accessible. This tutorial and its predecessors is the top-viewed tutorial from the Libraries with an average of 1000 views each month.

Unique Contibution

My contribution to the project was taking the lead on project management and instructional design, investigating creation tools, creating the CSS and HTML, and co-developing the assessment in Qualtrics.
